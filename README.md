# 🌿 주식회사 팔로우엔 - 영업부서 파이프라인

> 마케팅 실행사를 위한 영업 퍼널 관리 시스템

---

## 📌 프로젝트 개요

**주식회사 팔로우엔**의 영업팀을 위한 전문 파이프라인 관리 웹 애플리케이션입니다.
15개 영업 단계를 시각적으로 추적하고, 전환율을 실시간으로 분석하며, 팀의 영업 성과를 한눈에 확인할 수 있습니다.

## ✨ 주요 기능

### 1. 🚀 스플래시 화면
- 앱 로딩 시 팔로우엔 브랜딩 애니메이션
- 회사 로고 + 회사명 페이드인
- 1.5초 후 메인 화면 자동 전환

### 2. 📊 3가지 뷰 모드

#### 전체 파이프라인
- 15개 단계 카드 가로 스크롤
- 각 단계별 진행수 입력
- 실시간 전환율 계산
- 드래그앤드롭으로 순서 변경

#### 일별 추적
- 일별 확인 가능한 9개 단계만 큰 카드로 표시
  - 1번: 1차콜 (통화량)
  - 2번: 1차자료 발송
  - 3번: 1.5차자료 발송 (통계자료)
  - 5번: 1.5차자료 발송 (후기특자료)
  - 11번: 미팅
  - 12번: 팀장가망관리
  - 13번: 계약 완료
  - 14번: 2차 자료
  - 15번: 팀원 당일시트 이동

#### 퍼널 차트
- Chart.js 기반 시각적 차트
- 전체 통계 대시보드
- 단계별 전환율 분석

### 3. ✏️ 완전한 CRUD 기능
- **추가**: 플로팅 + 버튼으로 새 단계 추가
- **수정**: 각 카드의 연필 아이콘으로 수정
- **삭제**: 휴지통 아이콘으로 삭제
- **순서 변경**: 드래그앤드롭 (SortableJS)

### 4. 💾 데이터 영속성
- localStorage 자동 저장
- 브라우저 새로고침해도 데이터 유지
- 리셋 버튼으로 초기화 가능

## 🚀 사용 방법

### 1️⃣ 웹앱 실행
- 브라우저에서 `index.html` 파일 열기
- 또는 웹 서버에 배포

### 2️⃣ DB 투입량 입력
- 헤더 우측 "DB 투입량" 필드에 오늘의 DB 수 입력

### 3️⃣ 단계별 수치 입력
- **전체 파이프라인 탭**: 15개 모든 단계 입력
- **일별 추적 탭**: 일별 확인 가능한 9개 단계만 큰 입력창으로 표시

### 4️⃣ 단계 수정/추가/삭제
- **수정**: 카드의 연필 아이콘(✏️) 클릭
- **삭제**: 카드의 휴지통 아이콘(🗑️) 클릭
- **추가**: 우측 하단 플로팅 + 버튼 클릭
- **순서 변경**: 카드를 드래그앤드롭

### 5️⃣ 차트 분석
- **퍼널 차트 탭** 클릭
- 입력된 수치 기반으로 자동 시각화
- 전체 전환율 통계 확인
